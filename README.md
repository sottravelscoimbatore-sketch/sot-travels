# SOT Travels — Fixed Bootstrap Website

Upload all files from this folder to the website root.

## Folder structure
- HTML pages, `style.css`, `app.js`, `bootstrap.min.css` and `bootstrap.bundle.min.js` stay beside `index.html`.
- Put the logo at `images/1.png`.
- Put destination photos inside `images/destinations/`.
- Put vehicle photos inside `images/vehicles/`.

The Bootstrap CSS and JavaScript are included locally, so the mobile menu and enquiry modals work during local testing as well as after hosting.
